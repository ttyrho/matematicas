# Condicional lógico

| $p$ | $q$ | $p \rightarrow q$ |
| :-: | :-: | :---------------: |
|  0  |  0  |         1         |
|  0  |  1  |         1         |
|  1  |  0  |         0         |
|  1  |  1  |         1         |

**Observación**: La [[Proposición lógica|proposición]] $p \rightarrow q$ es falsa únicamente si $p$ es verdadera y $q$ es falsa.
